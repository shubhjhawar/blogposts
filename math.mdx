# Mathematical Formulas

This is an example of using mathematical formulas in MDX.

## Inline Math

You can include inline math expressions using single dollar signs like this: $y = mx + b$.

## Block Math

For displaying larger math expressions, you can use double dollar signs to create a block of math equations:

$$
```markdown
(a + b)^2 = a^2 + 2ab + b^2
```
$$

That's it!
